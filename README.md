# DEFINED Upload Package

This folder contains the current DEFINED web app files ready to upload to GitHub.

## Files

- `DEFINED_Home_Launcher.html`
- `DEFINED_Estimate_Studio.html`
- `DEFINED_Proposal_Studio.html`
- `DEFINED_Design_System.css`
- `DEFINED_Config.js`

## Preview States

The launcher supports internal preview states by URL:

- `?preview=free`
- `?preview=estimate`
- `?preview=suite`

Examples:

- `DEFINED_Home_Launcher.html?preview=free`
- `DEFINED_Home_Launcher.html?preview=estimate`
- `DEFINED_Home_Launcher.html?preview=suite`
